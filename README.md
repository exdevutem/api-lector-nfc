## API Lector NFC
Esta es una api simple que permite revisar si un tag NFC es válido y es miembro del club.

## Instalación
Para instalar la api, se debe clonar el repositorio y ejecutar los siguientes comandos:
```
pnpm install
pnpm run dev
```
